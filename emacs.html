1 Emacs 插件

1.1 YASnippet

1.1.1 snippet

新建 snippet
M-x yas-new-snippet key: C-c & C-n
查看 snippet
M-x yas-visit-snippet-file key: C-c & C-v

1.1.2 snippet 语法

语法类似与 TextMate, 并且可以内嵌 Emacs-lisp 代码. 更多细节参看YASnippet文档

1.2 org-mode

1.2.1 Link 相关

link 的格式为 \[\[link\]\[link-name\]\]

快捷键 操作
C-c l org-store-link
C-c C-l org-insert-link
C-c C-o org-open-at-point

1.2.2 编辑代码

快捷键 操作
<s 创建代码块
C-c ' 编辑代码块

1.3 目录操作

快捷键 功能
C-x d 打开目录 M-x dired

1.3.1 文件的删除复制重命名

快捷键 功能
d 加上待删除标记
u 取消标记
x 删除标记的文件
D 删除指定文件
C 复制
R 重命名
= 比较

2 Emacs 函数

  • 配置语法显示时的有用命令
    当前字符位置及描述
    (what-cursor-position &optional DETAIL) key: C-x = 如果加上前缀调用(C-u C-x =), 则显示字符的细节信息, 如字体, 语法, 编码等.
    显示当前位置face
    (M-x describe-face).

3 Emacs 快捷键

快捷键 操作
C-x k 关闭某个buffer
C-x s 保存
C-x c 退出
C-x 0  
C-x 1 当前buffer最大化
