1 \TeX 学习笔记

1.1 字符

1.1.1 字符分类

Category Meaning
0 Escape character
1 Beginning of group
2 Math shift
3 Alignment tab
5 End of line
6 Parameter
7 Superscript
8 Subscript
9 Ignored character
10 Space
11 Letter
12 Other character
13 Active character
14 Comment character
15 Invalid character
\catcode
修改字符的分类(Category), 如 \catcode`\{=1 把符号 { 的分类改为 1.

1.1.2 字符表示方式

  1. 可见字符

    直接可用键盘输入, 如 A, B, C 等.

  2. 不可见字符
    \char
    使用 \char<number> 可以得到任何你想要的字符, 这里 <number> 可以是从0到255的任意整数, 如 \char98 表示 b.
    ` (left quote)
    其后接任意字符或单字符控制序列表示该字符的 \TeX 内部码. 例如 \char`b 和 \char`\b 等价于 \char98.

1.2 \if... 命令

语法: \if<condition><true text>\else<false text>\fi

命令 说明
\ifnum<number1><relation><number2> compare two integers
\ifdim<dimen1><relation><dimen2> compare two dimensions
\if<token1><token2> test if character codes agree
\ifcat<token1><token2> test if category codes agree
\ifx<token1>token2> test if tokens agree
\iftrue, \iffalse always true or always false

多支条件语句: \ifcase<number><text for case 0>\or<textfor case 1>\or...\or<text for case n>\else<text for all other cases>\fi

\newif
新建 \if... 结构. 如 \newif\ifabc 会生成三个宏: \ifabc 用于测试条件真假, \abctrue 设置条件为真, \abcfalse 设置条件为假.

1.3 \def

语法: \def\newcs{replacement text}

在语法说明中, 大括号是必须的, 并不表示分组, 新的命令 \newcs 不包含大括号. 若要包含大括号, 可以如下定义 \def\newcs{{replacement text}}.

1.4 \expandafter, \noexpand

\expandafter<token>
先不展开其后面的宏, 并读取后面的内容, 然后再展开其后面的宏, 并插入在应有的位置.
\noexpand<token>
其展开是其后 token 自身.
